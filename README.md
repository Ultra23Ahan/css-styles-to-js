# Hello!

this is a simple css styles to js styles converter<br />
example:<br /><br />
input:<br />
```css
background-color: "green";```<br /><br />
output:<br />
```js
element.style.backgroundColor = "green";``<br /><br />

You can also change the elements name in case it is nto set to "element"<br />
thats it!
